# Themes

This `themes/` folder contains themes files.

* `_theme.less`
* `_admin.less`

This folder is very project-specific and is likely to be non-existent on many projects.

## Structure

```
styles/
|
|– themes/
|   |– _theme.less        # Default theme
|   |– _admin.less        # Admin theme
|   …                     # Etc.
```
